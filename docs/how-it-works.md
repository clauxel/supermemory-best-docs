# How It Works - Memory Hygiene Audit

Memory Hygiene Audit scans AI agent memory and repo instructions for stale facts, context bloat, unsafe retention, contradictions, and poisoning risk.

## Paste a memory sample

Submit redacted memory, repo instructions, handoff notes, or agent rules.

## Classify source and risk

Separate durable facts from temporary state, personal preference, tool output, and unsafe instructions.

## Generate cleanup guidance

Review contradictions, expired notes, retention advice, and recommended compact replacement.

## Export policy evidence

Use a paid report for team review, cleanup diffs, and agent-readable JSON.
