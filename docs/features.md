# Features - Memory Hygiene Audit

Memory Hygiene Audit scans AI agent memory and repo instructions for stale facts, context bloat, unsafe retention, contradictions, and poisoning risk.

## Source-aware memory audit

Classify memory as repo fact, user preference, temporary incident note, tool output, external text, or unsafe instruction.

## Context bloat cleanup

Identify stale build commands, repeated handoff notes, outdated deployment warnings, and expired rules.

## Poisoning risk review

Flag instructions that try to override policy, store secrets, preserve personal dossiers, or trust untrusted tool output.

## AGENTS.md and CLAUDE.md audit

Review agent instruction files for scope, conflicts, freshness, and operational clarity.

## Retention policy builder

Turn memory findings into compact agent-readable retention and cleanup guidance.

## Team evidence exports

Unlock full reports, cleanup diffs, owner notes, and JSON policy for shared review.
