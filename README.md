## Boundaries

- The scanner supports governance and review; it does not guarantee perfect memory safety.
- Teams should review outputs before adopting them as security, compliance, or production policy.
- Raw secrets and unnecessary personal data should not be submitted.
- Memory Hygiene Audit is independent and is not affiliated with or endorsed by SupermemoryAI or any upstream project.
